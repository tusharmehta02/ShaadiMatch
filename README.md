# ShaadiMatch
Shadi Matches

This project fetches information about shadi matches and stores locally about user action i.e. Accept/Decline.

## Following dependencies are used:
1.MVVM<br />
2.Room<br />
3.Dagger 2<br />
4.Material components<br />
5.Jetpack components
